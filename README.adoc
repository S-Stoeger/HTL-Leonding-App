== OmniaL

=== Einführung
Das Projekt HTL-Leonding-App handelt von einer Anwendung, auf welcher es Schülern ermöglicht Räume zu reservieren.
Auch der Verleih von Equipment wird hier festgehalten bzw. kann auch durchgeführt werden.
